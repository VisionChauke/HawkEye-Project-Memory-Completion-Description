# HawkEye-Project-Memory-Completion-Description
In this lab, analysts examine captured network traffic to uncover how the attacker gained access, maintained control, and exfiltrated stolen credentials. Using tools such as Wireshark, Brim, CyberChef, Apackets, MaxMind GeoIP, and VirusTotal, participants will:
--------
. Using tools such as Wireshark, Brim, CyberChef, Apackets, MaxMind GeoIP, and VirusTotal, participants will:
•	Analyze PCAP files to identify suspicious traffic patterns
•	Detect Indicators of Compromise (IoCs)
•	Decode and reconstruct exfiltrated data
•	Identify stolen credentials and command-and-control (C2) communication
•	Trace attacker infrastructure using IP reputation and geolocation services

Ideal for SOC analysts, incident responders, blue team professionals, and cybersecurity students seeking practical experience in network-based malware investigations.
